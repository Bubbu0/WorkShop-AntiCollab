Phần này tổng hợp các bài viết kỹ thuật mà chúng em đã thực hiện trong quá trình thực tập, tập trung vào kiến trúc AWS, DevOps và tích hợp Generative AI. Những chia sẻ này được đóng góp cho cộng đồng [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj).

###  [Blog 1 - Tìm hiểu về Amazon Q: Liệu AI có thể hỗ trợ toàn bộ vòng đời phát triển phần mềm?](3.1-Blog1/)
Khám phá cách Amazon Q chuyển đổi Vòng đời Phát triển Phần mềm (SDLC) bằng cách đóng vai trò như một trợ lý thông minh hỗ trợ viết code, debug và kiến trúc, vượt xa khả năng tạo code đơn thuần để cung cấp sự hỗ trợ toàn diện xuyên suốt quy trình phát triển.

###  [Blog 2 - Triển khai Internal DNS cho Internet-facing Load Balancer trên AWS bằng Kiến trúc Hướng sự kiện](3.2-Blog2/)
Trình bày giải pháp hướng sự kiện (event-driven) mạnh mẽ để giải quyết các thách thức về phân giải DNS cho Internet-facing Load Balancer bằng cách tự động hóa quản lý Private Hosted Zone với AWS Lambda, EventBridge và Amazon Route 53.

###  [Blog 3 - Tự động hóa Deployment lên Amazon ECS Express Mode bằng GitHub Actions](3.3-Blog3/)
Thảo luận về việc triển khai quy trình CI/CD an toàn bằng cách tích hợp GitHub Actions với Amazon ECS Express Mode, sử dụng OIDC để xác thực không cần mật khẩu (passwordless) nhằm tuân thủ nghiêm ngặt Nguyên tắc đặc quyền tối thiểu (Least Privilege).
